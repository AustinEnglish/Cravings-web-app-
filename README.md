# Cravings

Ever wonder what the top food places in your city are? Its never been easier with cravings!

## Getting Started

Clone and download from github.

### Prerequisites

  React required. Install all dependencies to run. API keys are required.

```
- zamato api
- fourSquare api
- zip 1 api
```
